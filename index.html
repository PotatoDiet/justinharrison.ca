Projects

  • WaniKani Tracker (2020) - Firefox webextension for tracking correct / incorrect answers and starting time of WaniKani review sessions.
  • GPA Calculator (2018) - Haskell GPA calculator which uses YAML to represent courses.
  • UofT Prof Ratings (2018, 2020) - Firefox webextension which embeds ratemyprofessors.com info onto UofT's timetable.
  • Anki Chinese Helper (2017 - 2018) - Helper add-on for Anki 2.1 to make studying Chinese more efficient.
  • diceware-gen (2017, 2020) - Rust-based program for generating random words. You pass it some natural N, and it spits out N random words.
  • forthan (2015) - Proof of concept forth interpreter built with C++11.
  • neapolis (2015) - C++11 Neocities client.
  • autoplete (2015 - 2016) - Autocompletion for HTML input elements.
  • gemtracker (2015) - C++11 program for finding the daily downloads count of arbitrary gems from rubygems.org.
  • dacado (2015) - Java program for embeding binary data into images. Useful for storing files on cloud services with cheap photo storage.
  • Image Board (2014) - Rails 4 webapp to share images. Includes tag-based searching, user classes, and comments.
  • AverageFrame (2014) - Java program for creating the average frame of a video.
  • Teamspeak Ruby Client (2014 - 2017) - Ruby interface for TeamSpeak 3's server query API.
  • Documentaries (2014 - 2015) - Rails 4 webapp to share videos, with a focus on documentaries.
  • OpenWorld (2014, 2017) - Unfinished Unity game developed in collaboration with this terrible man.
  • ActionScript2 Classes (2013) - Small collection of ActionScript classes to be used with CryEngine 3 and Scaleform. Definitely of no use to anyone, but this is one of the first codebases I ever released publically when I was first learning how to program.
