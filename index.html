Projects

  • WaniKani Tracker (2020) - Firefox webextension which allows you to track WaniKani review session statistics.
  • GPA Calculator (2018) - GPA calculator for a single semester built using Haskell.
  • UofT Prof Ratings (2018, 2020) - Firefox webextension which adds ratemyprofessors.com info to UofT's timetable.
  • Anki Chinese Helper (2017 - 2018) - Helper add-on for Anki 2.1 to make studying Chinese simpler.
  • diceware-gen (2017, 2020) - A simple diceware generator built using Rust and EFF's long wordlist.
  • forthan (2015) - Very basic forth interpreter built with C++11.
  • neapolis (2015) - C++11 wrapper for the Neocities API.
  • autoplete (2015 - 2016) - Browser-based input autocompletion built with TypeScript.
  • gemtracker (2015) - C++11 program for finding the daily downloads count for arbitrary gems from rubygems.org.
  • dacado (2015) - Java program for encoding / decoding arbitrary data into images.
  • Image Board (2014) - Rails 4 webapp to share images. Includes tag-based searching, user classes, and comments.
  • AverageFrame (2014) - Java program for creating the average frame of a video.
  • Teamspeak Ruby Client (2014 - 2017) - Ruby interface for TeamSpeak 3's server query API.
  • Documentaries (2014 - 2015) - Rails 4 webapp to share videos, with a focus on documentaries.
  • OpenWorld (2014, 2017) - Unfinished Unity game developed in collaboration with this terrible man.
  • ActionScript2 Classes (2013) - Small collection of ActionScript classes to be used with CryEngine 3 and Scaleform. Definitely of no use to anyone, but this is one of the first codebases I ever released publically when I was first learning how to program.
